 # iFit
  
  ## Descrição
  App fitness para registro de treinos e progressão de carga
  
  ## Tecnologias
  - Flutter
  - Firebase (Auth, Firestore)
  - [Outros packages importantes]
  
  ## Setup
  1. Clone o repositório
  2. `flutter pub get`
  3. Configure Firebase (adicionar google-services.json e GoogleService-Info.plist)
  4. `flutter run`
  
  ## Estrutura do Projeto
  - `/lib/core` - Constantes, utils
  - `/lib/data` - Models e services
  - `/lib/presentation` - UI (screens, widgets, controllers)
